Diseña la interfaz de un sistema web para una corredora de seguros llamada Virgo Corredores de Seguros.

La marca debe transmitir confianza, cercanía, claridad y acompañamiento. No debe sentirse como una aseguradora fría o corporativa, sino como un socio confiable y humano.

## Objetivo del producto
Crear una plataforma web con tres áreas principales:
1. Canal de autogestión para clientes actuales.
2. Portal privado del asegurado con login.
3. Sección comercial para captar nuevos prospectos.

## Estilo visual
- Moderno, limpio y profesional.
- Muy claro visualmente.
- Uso de blancos y buena respiración.
- Inspirado en la identidad visual existente de Virgo.
- Paleta basada en verdes, turquesas, lima y tonos luminosos del logo.
- Evitar estilos oscuros, duros o excesivamente corporativos.

## Pantallas a diseñar
1. Home
2. Página Seguros para ti
3. Página Seguros para empresas
4. Página Seguro para mascotas
5. Página Cotiza tu seguro
6. Página de contacto
7. Login portal del asegurado
8. Dashboard del asegurado
9. Vista de pólizas activas
10. Vista de historial de gestiones
11. Vista de trámites pendientes
12. Vista de estado de solicitud

## Elementos que debe incluir
- Hero principal con propuesta de valor clara.
- CTA visibles: Cotiza tu seguro, Portal del asegurado, Contáctanos.
- Bloques de productos/servicios.
- Beneficios de Virgo: atención personalizada, respuesta rápida, asesoría experta, soluciones a medida.
- Sección de confianza con logos de clientes o aliados.
- Formularios claros, amables y simples.
- Dashboard privado muy legible y ordenado.
- Navegación clara entre zona pública y privada.

## Reglas UX
- Mobile first.
- Formularios breves.
- Jerarquía clara.
- Estados vacíos y mensajes de confirmación.
- Diseño pensado para usuarios no técnicos.
- Priorizar usabilidad sobre complejidad visual.

Genera una propuesta de interfaz coherente, elegante, funcional y preparada para convertirse luego en un producto real.
